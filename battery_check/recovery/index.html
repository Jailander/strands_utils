Recovery procedure

Follow these instructions in the case the robot is unable to find its charging station and its battery level is low.

Switch to free run mode

The free run mode switches off the motor control and allows to move the robot around by hand. The free run mode can be switched on using the base control panel located at the lower starboard of the robot.

Use the grey knob to navigate the control menu. Go 'back' to the menu's root, open the Swithing menu and then set the Free run to ON.

Push the robot onto the charging station.

Grab the robot at its monitor and move it onto the charging station. Once the robot reaches the charging station, it will start to charge automagically. The charging process is indicated by a yellow LED on the charger base. If the LED switches on, you can leave the robot as it is - it will run self diagnostics and try to resume patrolling. If the LED does not switch on despite the robot is positioned on the charger, switch the robot off by turning the keys next to the LED control panel.
